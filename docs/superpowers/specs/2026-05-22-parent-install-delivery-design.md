# 家长安装交付优化设计

## 目标

把“舞动成长 AI 学习助手｜免费本地版”从开发者可运行项目，补成普通家长也能顺着说明完成准备和启动的交付形态。

本次改动聚焦 Windows 家长入口：

- 用页面说明降低第一次安装门槛。
- 用 `install.bat` 准备电脑端本地 AI。
- 用 `start.bat` 检查启动环境并打开学习助手。
- 在首页和 README 给出清楚入口。

## 设计选择

采用“家长安装引导页 + 双脚本分工”。

`install.bat` 只处理电脑端 AI 的准备工作：检查 Ollama、检查并下载 `qwen3:1.7b`。`start.bat` 只处理学习助手启动：检查 Node.js 和 npm、启动 Next.js 项目、打开首页。这样家长能把“先准备 AI”和“再打开学习助手”分成两个动作理解。

## 页面结构

### 首页入口

首页主操作区新增“家长安装引导”按钮，链接到 `/install-guide`。

首页保持当前正式 AI 学习机产品感，不把安装细节继续堆进首屏正文。安装说明集中到专门页面。

### 家长安装引导页

新增 `/install-guide` 页面，延续现有中文页面与高级简洁视觉语言，用家长说明书口吻组织内容。

页面内容分成四块：

1. 先说明安心点
   - 本工具不接云端 AI API。
   - 不产生 AI API 调用费用。
   - 电脑端 AI 在本机运行。
2. Windows 电脑准备步骤
   - 第一步双击 `install.bat`，准备 Ollama 与轻量模型。
   - 第一次需要下载 `qwen3:1.7b`。
   - 没有独立显卡也可以先尝试轻量模型，速度取决于电脑配置。
3. 启动步骤
   - 第二步双击 `start.bat`。
   - 如果缺少 Node.js 或 npm，按中文提示安装 Node.js 后重新双击。
   - 使用电脑端 AI 时保持 Ollama 运行。
4. 设备不合适时的退路
   - 如果电脑跑不动，仍可使用学习工具箱模式。
   - 手机和平板默认先使用学习工具箱模式，避免家长以为必须安装电脑软件。

页面 CTA 保持少而明确，可提供返回首页、查看设备检测或进入学习工具箱相关入口。

## Windows 脚本

### `install.bat`

职责：

- 检查当前电脑是否能调用 `ollama`。
- 找不到 Ollama 时，显示普通家长能看懂的中文提示，说明需要先到 Ollama 官网下载安装，安装后重新运行脚本。
- 找到 Ollama 后检查本机是否已有 `qwen3:1.7b`。
- 未发现模型时执行 `ollama pull qwen3:1.7b`。
- 下载成功后提示电脑端 AI 已准备好，可以继续启动学习助手。
- 下载失败时给出友好提示，提醒检查网络、Ollama 是否正常运行后重试。

边界：

- 不负责安装 Node.js。
- 不负责启动 Next.js 项目。
- 不负责下载更大的推荐或增强模型。

### `start.bat`

职责：

- 检查 Node.js 与 npm 是否可用。
- 如果缺少任一项，显示中文提示：
  - 当前电脑缺少 Node.js 运行环境，需要先安装 Node.js。
  - 提示去 Node.js 官网下载。
  - 提醒安装完成后重新双击 `start.bat`。
- 运行环境齐全时启动项目。
- 自动打开 `http://localhost:3000`。
- 提醒家长若要使用电脑端 AI，请保持 Ollama 运行。

边界：

- 不下载 Ollama 模型。
- 不替代 `install.bat` 的本地 AI 准备流程。

启动实现优先保持简单稳定：让脚本在项目根目录运行，复用项目现有 npm 启动命令。若依赖尚未安装，脚本应先保证启动前依赖可用或给出明确提示，避免直接把命令行错误暴露给家长。

## README 更新

README 新增“普通家长安装流程”段落，优先讲两步：

1. 先准备电脑端 AI：运行 `install.bat`。
2. 再启动学习助手：运行 `start.bat`。

该段落补充：

- 脚本各自负责什么。
- 缺少 Ollama 时如何处理。
- 缺少 Node.js 时如何处理。
- `qwen3:1.7b` 是普通家庭电脑优先尝试的轻量模型。
- 学习工具箱模式是低配置设备的可用退路。

README 继续保留开发者本地运行命令，和家长流程并列而不互相替代。

## 错误提示原则

- 所有新增页面与脚本文案默认用中文。
- 先告诉家长“接下来做什么”，再解释原因。
- 避免把 `PATH`、包管理器、端口占用等开发术语直接暴露成主提示。
- 对必要名称保留原名，例如 Ollama、Node.js、`qwen3:1.7b`、`install.bat`、`start.bat`。
